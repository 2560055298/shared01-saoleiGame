HGE::Resource_EnumFolders

根据给定的通配符枚举文件夹。 

char *Resource_EnumFolders(
  const char *wildcard = 0
);

参数

wildcard
要开始一个新的枚举，设置包括特殊字符“?”和“*”在内的相对或绝对的通配符， 要继续前一次的枚举请使用0。

返回值

返回找到的下一个文件夹名，如果没有文件夹或遍历到结尾则返回0。

备注

Resource_EnumFolders返回不包含文件路径的简单文件夹名。 并且它会忽略文件，要枚举文件，请使用Resource_EnumFiles。

要枚举某位置的所有文件夹请使用“*”通配符。 例如Resource_EnumFolders("data\*")将枚举所有位于程序所在目录中的data文件夹中的所有文件夹。

需要

头文件: hge.h
库文件: hge.lib

See also

Resource_EnumFiles, Resource_MakePath
