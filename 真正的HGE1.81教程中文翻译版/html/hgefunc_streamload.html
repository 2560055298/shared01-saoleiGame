HGE::Stream_Load

从内存，资源包或磁盘中载入一个压缩音频流(Compressed Audio Stream)。 

HSTREAM Stream_Load(
  const char *filename,
  DWORD size = 0
);

参数

filename
流文件名称。
size
如果此参数不为0，那么他表示存有已知格式流媒体的内存块大小。 而参数filename则被认为是指向该内存块的指针。

返回值

成功则返回可供Stream_Play使用的流句柄。 否则返回0。

备注

支持 MP1, MP2, MP3, OGG， WAV等文件格式。

如果filename是一个相对路径，将首先在所有资源包中使用Resource_AttachPack 来检索并加载流文件，然后在程序文件夹中检索。 如果filename是一个绝对路径，流文件将直接被从磁盘加载。

如果函数调用失败，你可以使用System_GetErrorMessage函数来获得错误信息。 如果设置了系统标记HGE_LOGFILE，错误信息将被写入日志。

当流不再被使用时，需要使用函数Stream_Free来删除它。

需要

头文件: hge.h
库文件: hge.lib

另见

Stream_Free, Stream_Play
