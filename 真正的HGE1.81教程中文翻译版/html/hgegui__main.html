hgeGUI帮助类

hgeGUI是一个用来创建和管理用户界面的HGE帮助类。

hgeGUI方法

构造函数 创建并初始化hgeGUI对象。
  
AddCtrl 向GUI中增加一个控件。
DelCtrl 从GUI中移除一个控件。
GetCtrl 返回指向根据标识确定的控件的指针。
MoveCtrl 移动一个控件。
ShowCtrl 显示或隐藏一个控件。
EnableCtrl 使一个控件被激活或失效。
  
SetNavMode 设置GUI导航模式。
SetCursor 设置鼠标指针的精灵。
SetColor 设置所有GUI控件的颜色。
SetFocus 设置键盘输入焦点。
GetFocus 返回当前键盘输入焦点。
Enter 启动进入GUI动画。
Leave 启动离开GUI动画。
Reset 将GUI设置为一个默认的状态。
Move 将所有的GUI控件移动到一个新的位置。
  
Update 更新GUI并处理用户输入。
Render 将GUI渲染到屏幕上。

相关信息

hgeGUIObject 自定义GUI控件的抽象基类。
  
hgeGUIText 文本GUI控件类。
hgeGUIButton 按钮GUI控件类。
hgeGUISlider 滑动条GUI控件类。
hgeGUIListbox 列表框GUI控件类。
  
Macros 默认控件管理宏。
