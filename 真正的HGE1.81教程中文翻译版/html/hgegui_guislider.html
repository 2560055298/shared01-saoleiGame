hgeGUISlider类

hgeGUISlider是一个滑动条GUI控件类。 要创建一个hgeGUISlider对象可以使用如下构造函数： 

hgeGUISlider(
  int       id,
  float     x,
  float     y,
  float     w,
  float     h,
  HTEXTURE  tex,
  float     tx,
  float     ty,
  float     sw,
  float     sh,
  bool      vertical = false
);

参数

id
已创建的控件整型标识。
x
滑动条的左上角X坐标。
y
滑动条的左上角Y坐标。
w
滑动条的宽度。
h
滑动条的高度。
tex
纹理句柄，保存有滑动条的图像。根据滑动条模式的不同图像也会有不同的解释方式。 详情参考SetMode函数的描述。
tx
滑动条图像左上角纹理X坐标，以像素为单位。
ty
滑动条图像左上角纹理Y坐标，以像素为单位。
sw
滑动条图像的宽度。
sh
滑动条图像的高度。
vertical
指明滑动条是否垂直。

其他方法

SetMode 设置滑动条的行为模式。
SetValue 设置当前滑动条的位置。
GetValue 返回当前滑动条的位置。

需要

头文件: hgeguictrls.h
库文件: hgehelp.lib
