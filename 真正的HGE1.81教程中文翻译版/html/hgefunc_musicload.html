HGE::Music_Load

从内存，资源包或者磁盘文件中载入一段音乐。 

HMUSIC Music_Load(
  const char *filename,
  DWORD size = 0
);

参数

filename
音乐文件名。
size
如果此参数不是0，则此参数指明包含任意有效格式音乐的内存块的大小。 而参数filename则被认为是指向该内存块的指针。

返回值

成功则返回可供Music_Play函数使用的音乐句柄。 否则返回0。

备注

支持MO3, IT, XM, S3M, MTM, MOD 和 UMX等音乐文件格式。

如果filename是一个相对路径，则会使用Resource_AttachPack先从所有资源包中检索并 附加该文件，然后是程序目录。 如果filename是一个绝对路径，将直接把音乐文件通过指定的路径从磁盘载入。

如果函数调用失败，可是通过调用System_GetErrorMessage函数获得错误信息。 如果设置了系统标记HGE_LOGFILE，错误信息将被写入日志文件。

当音乐不再被使用时，必须使用Music_Free函数删除它。

需要

头文件: hge.h
库文件: hge.lib

另见

Music_Free, Music_Play
