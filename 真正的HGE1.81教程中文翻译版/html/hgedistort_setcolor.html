hgeDistortionMesh::SetColor

为指定的扭曲节点设置颜色和透明度。 

void SetColor(
  int col,
  int row,
  DWORD color
);

参数

col
要改变的节点的列。列数字从0开始。
row
要改变的节点的行。行数字从0开始。
color
新的颜色值。

备注

颜色参数中包含了透明度值。详情请参考Hardware color format部分。 颜色叠加到纹理数据上的方式取决于混合模式。 更多信息请参考Blending modes。

需要

头文件: hgedistort.h
库文件: hgehelp.lib

另见

GetColor, SetZ, SetDisplacement, SetBlendMode, 硬件颜色格式
