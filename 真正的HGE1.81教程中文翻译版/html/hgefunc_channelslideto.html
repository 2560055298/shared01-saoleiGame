HGE::Channel_SlideTo

滑动一个通道的音量，均衡或音高。 

void Channel_SlideTo(
  HCHANNEL channel,
  float time,
  int volume,
  int pan = -101,
  float pitch = -1
);

参数

channel
指定的音频通道句柄。
time
滑块设定的时间点。
volume
新的音量值，0~100。-1代表保持当前值。-2代表见渐减到0并停止该通道。
pan
新的均衡值， -100~100. -101代表保持当前值。可选。
pitch
新的音高倍数， 1.0 表示不做修改。 -1.0代表保持当前值。可选。

备注

多次调用Channel_SlideTo你可以同时得到不同时间具有不同音效的通道。

需要

头文件: hge.h
库文件: hge.lib
