电源管理

很多玩家使用移动电脑，靠电池来供电。 因此出于礼貌的说，你的游戏最好遵循电源相关条例。 下面是一些非常重要的建议。

限制FPS

对于绝大多数游戏而言，没有必要使用CPU的全部计算资源。 要降低电源损耗，延长电源使用时间，需要使用HGE_FPS系统标记来限制游戏的FPS： 

hge->System_SetState(HGE_FPS, 50);

检测低电量的电源

使用HGE_POWERSTATUS系统标记来显示电源状态：

  • HGEPWR_UNSUPPORTED值表示OS不支持电源状态报告，我们也没辙了。
  • HGEPWR_AC值表示连接的是直流AC电源，那我们就没啥好担心的了。
  • Values 0-100反映出当前电池的电量，当然也就是说正在使用电池啦。

例如，检测电池电量是否小于5%可以使用如下代码： 

int pstatus = hge->System_GetState(HGE_POWERSTATUS);

if(pstatus != HGEPWR_UNSUPPORTED)
{
	if(pstatus != HGEPWR_AC && pstatus < 5)
	{
		//用一条信息或者警告图标提示用户
		//保存用户数据
	}
}

如果你检测到电池电量小于5%，建议做如下操作：

  • 通过信息活图标提示玩家。
  • 保存所有玩家数据和进度以防数据丢失
