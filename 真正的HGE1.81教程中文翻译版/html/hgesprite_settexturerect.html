hgeSprite::SetTextureRect

设置供精灵使用的纹理区域。 

void SetTextureRect(
  float x,
  float y,
  float w,
  float h,
  bool adjSize = true
);

参数

x
纹理区域的左上角X坐标。
y
纹理区域的左上角Y坐标。
w
纹理区域的宽度。
h
纹理区域的高度。
adjSize
如果为true, 精灵的维数将根据新纹理区域进行调整，如果为false 精灵的维数将不变。可选。

需要

头文件: hgesprite.h
库文件: hgehelp.lib

另见

GetTextureRect, SetTexture, GetTexture
