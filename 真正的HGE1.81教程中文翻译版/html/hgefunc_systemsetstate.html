  bool value

void System_SetState(
  hgeStringState state,

);

参数

state
System states枚举类型中的成员项。 指定系统标记。
value
要设置的系统标记的新值

备注

用此函数可以设置很多所需的内部系统标记，例如屏幕分辨率，窗口标题，日志文件名称等。 虽然绝大部分系统标记都有默认值，但是需要注意的是在System_Start被调用之前 必须设置HGE_FRAMEFUNC的值。

更多信息请参考System states部分。
