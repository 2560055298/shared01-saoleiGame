hgeSprite::RenderEx

将缩放并旋转后的精灵渲染到屏幕上。 

void RenderEx(
  float x,
  float y,
  float rot,
  float hscale = 1.0f,
  float vscale = 0.0f
);

参数

x
要渲染到屏幕上的位置的X坐标。
y
要渲染到屏幕上的位置的Y坐标。
rot
精灵旋转的弧度。
hscale
水平缩放因子，1.0代表默认大小，如果省略将不进行缩放。
vscale
垂直缩放因子，1.0代表默认大小，如果为0或者省略hscale将被用于水平和垂直缩放。

备注

缩放，旋转和位移操作运算都与精灵的锚点有关。 详细请参考SetHotSpot

需要

头文件: hgesprite.h
库文件: hgehelp.lib

另见

Render, RenderStretch, Render4V
