HGE::Gfx_SetTransform

为所有即将被渲染的图形设置全局转换（global transformation）。 

void Gfx_SetTransform(
  float x = 0,
  float y = 0,
  float dx = 0,
  float dy = 0,
  float rot = 0,
  float hscale = 0,
  float vscale = 0
);

参数

x
中心点X坐标。
y
中心点Y坐标。
dx
中心点X位移。
dy
中心点Y位移。
rot
旋转角度（单位：弧度）。
hscale
水平缩放因子。
vscale
垂直缩放因子。

备注

转换将以如下顺序进行：首先缩放，然后旋转，最后位移。

如果所有参数被设置为0或者省略，全局转换将被重置。 如果渲染目标转换为其他目标，全局转换也将被重置。

Gfx_SetTransform会冲掉所有之前积累的图形元（graphic primitives）。

需要

头文件: hge.h
库文件: hge.lib
