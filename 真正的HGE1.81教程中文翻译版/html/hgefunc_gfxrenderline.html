HGE::Gfx_RenderLine

渲染一条线。 

void Gfx_RenderLine(
  float x1,
  float y1,
  float x2,
  float y2,
  DWORD color = 0xFFFFFFFF,
  float z = 0.5f
);

参数

x1
起始X坐标。
y1
起始Y坐标。
x2
终止X坐标。
y2
终止Y坐标
color
可选的颜色值。默认为0xFFFFFFFF。
z
可选的Z顺序。默认为0.5f。

被指

Gfx_RenderLine必须在Gfx_BeginScene和Gfx_EndScene 函数之间调用。

Gfx_RenderLine省略了线的最后一个点，因此你可以在不必做额外的坐标检测的前提下绘制连接线。 颜色参数包含透明度值，详情请参考硬件颜色格式部分。 如果没有使用Z缓冲，Z顺序将被忽略。

考虑到效率问题Gfx_RenderLine积攒连续的几行(文字)， 然后使用一个DirectX调用将其渲染出来。 绝大多数时候你不会发现其中的不同，并且也没有必要去处理这些批处理操作。 但是要保证准确和安全，建议还是使用原始类型组织好图形并尽可能多的一次渲染多条线。

需要

头文件: hge.h
库文件: hge.lib
