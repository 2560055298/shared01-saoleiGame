Visual Studio .NET

1. 设置头文件和库文件路径

打开Tools->Options->Projects->VC++ Directories, 将HGE的头文件“include”目录加入并移到顶部。:


在相同的标签下从Show directories for下拉列表中选择Library files 加入HGE的库文件“lib\vc”目录并移到顶部

:


点击OK.

现在你可以不必指出准确的路径就能在你的工程中使用HGE游戏引擎了。这步只需配置一次，以后不必在配置。

2. 创建一个空的Win32 Application工程

打开File->New...->Projects对话框标签, 选择Win32 Project,输入你的工程名和存放路径:


点击OK.

在下一个窗口，选择Application Settings。确保Application Type下的Windows Application被设置。 然后勾选Additional Options下的Empty project，如果你想要软件帮你生成一个简单的文件结构可以不选中它：


点击Finish。哐啷! 你的工程创建完毕。

3. 向工程中加入库文件

打开Project->Properties...->Linker->Input向Additional Dependencies中加入如下库文件:

  • hge.lib - HGE系统层DLL加载库
  • hgehelp.lib - HGE帮助类库文件

如果你不调用任何Windows函数，你只需加载这两个库文件。


然后从Configuration 下拉列表中选择Release重复上面的操作。

点击OK。

4. 开始编码!

包含hge.h头文件来访问HGE功能函数. 参考教学指导 部分了解初级编码的例子.
