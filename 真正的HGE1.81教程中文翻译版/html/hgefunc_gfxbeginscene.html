HGE::Gfx_BeginScene

开始渲染图像。 

bool Gfx_BeginScene(
  HTARGET target = 0
);

参数

target
可选的渲染目标句柄。 如果是0或者省略将使用屏幕表面。

返回值

成功则true，否则返回false。

备注

Gfx_BeginScene必须在所有HGE渲染函数和帮助类的渲染之前被调用。

如果此函数调用失败，可以使用System_GetErrorMessage函数获得错误信息， 另外如果设置了HGE_LOGFILE标记，错误信息将被写入日志。

要终止渲染你必须调用Gfx_EndScene函数。 嵌套的调用Gfx_BeginScene / Gfx_EndScene是非法的。

需要

头文件: hge.h
库文件: hge.lib

另见

Gfx_EndScene, Target_Create
