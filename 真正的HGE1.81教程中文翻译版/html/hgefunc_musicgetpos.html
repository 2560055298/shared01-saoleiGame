HGE::Music_GetPos

返回一段音乐样本列的当前位置。 

void Music_GetPos(
  HMUSIC music,
  int *order,
  int *row
);

参数

music
音乐句柄。
order
指向用语存放该音乐当前顺序的int型指针。
row
指向用语存放该音乐当前列的int型指针。

备注

可以使用Music_GetPos获得以orders和rows形式存放的音乐播放的位置。 在调用Music_Play函数后还可以使用 Channel_GetPos来获得播放位置(以秒的形式)。

备注

头文件: hge.h
库文件: hge.lib
