HGE 顶点（Vertex）结构

hgeVertex结构用来描述构成HGE图形元（graphic primitives）的顶点。 

struct hgeVertex
{
  float  x, y;
  float  z;
  DWORD  col;
  float  tx, ty;
};

成员

x
屏幕X坐标
y
屏幕Y坐标
z
顶点的Z序（Z-order）范围从0.0到1.0
如果使用了Z缓冲，此值将被忽略。
col
顶点颜色和不透明度。
tx
纹理X坐标范围从0.0到1.0。
ty
纹理Y坐标范围从0.0到1.0。

备注

如果你为纹理坐标指定了0.0到1.0以外的值，纹理数据将会在每个整数点处重复，例如，可以以此轻松地实现rotozoom效果。
