hgeResourceManager Constructor

要创建并初始化一个hgeResourceManager对象可以使用如下构造函数： 

hgeResourceManager(
  const char *scriptname = 0
);

参数

scriptname
资源脚本文件名，如果=0或者省略，将不使用任何脚本。

备注

虽然管理复杂资源需要创建一个资源脚本，但是你仍然可以在不创建该脚本的情况下使用hgeResourceManager。 这样将允许你通过对应的文件名称来确定未处理资源，纹理，声效，音乐的地址。

所有通过hgeResourceManager对象加载的资源都将通过hgeResourceManager析构函数从内存中删除。

资源脚本文件通过Resource_Load函数加载，因此可以使用所有该对象的特性。

需要

头文件: hgeresource.h
库文件: hgehelp.lib
