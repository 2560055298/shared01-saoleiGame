HGE::Gfx_RenderQuad

渲染一个quad。 

void Gfx_RenderQuad(
  const hgeQuad *quad
);

参数

quad
指向hgeQuad的指针，描述要被渲染的quad。

备注

Gfx_RenderQuad函数必须在Gfx_BeginScene和Gfx_EndScene 函数之间调用。

Quad是基本的HGE图形元（graphic primitives）。更多quad的信息请参考Quad结构体部分。 如果quad的tex字段是0，白色将被用于纹理数据。 还需要注意的事quad的四个顶点都可以有它们自己的颜色，透明度和Z顺序(深度)。 Quad不需要做任何额外的屏幕和纹理坐标优化便可以自动调整与quad的接合渲染。

考虑到效率问题Gfx_RenderQuad积攒连续的若干含有相同的纹理和混合模式的quad并使用一个DirectX调用来渲染它们。 绝大多数时候你不会发现其中的不同，并且也没有必要去处理这些批处理操作。 但是为了保证准确和安全，建议还是通过纹理和混合模式来组织好你的Gfx_RenderQuad函数调用。
