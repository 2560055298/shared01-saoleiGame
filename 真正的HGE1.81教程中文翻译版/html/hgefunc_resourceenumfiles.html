HGE::Resource_EnumFiles

根据给定的通配符枚举文件。 

char *Resource_EnumFiles(
  const char *wildcard = 0
);

参数

wildcard
要开始一个新的枚举，设置包括特殊字符“?”和“*”在内的相对或绝对的通配符， 要继续前一次的枚举请使用0。

返回值

返回找到的下一个文件名，如果没有文件或遍历到结尾则返回0。

备注

Resource_EnumFiles返回不包含文件路径的简单文件名。 并且它会忽略文件夹，要枚举文件夹，请使用Resource_EnumFolders。

要枚举文件夹中的所有文件请使用“*”通配符。 例如Resource_EnumFiles("data\\*")将枚举所有位于程序所在目录中的data文件夹中的所有文件。

需要

头文件: hge.h
库文件: hge.lib

另见

Resource_EnumFolders, Resource_MakePath
