hgeResourceManager::GetDistortionMesh

返回指向一个已命名扭曲网格的指针。 

hgeDistortionMesh *GetDistortionMesh(
  const char *name
);

参数

name
扭曲网格名称。

返回值

如果成功，返回指向hgeDistortionMesh对象的指针，否则返回0。

备注

要通过hgeResourceManager使用扭曲网格，必须在资源脚本中定义它。

如果扭曲网格没有被创建，将先创建它。否则将返回指向已存在扭曲网格句柄的指针。

需要

头文件: hgeresource.h
库文件: hgehelp.lib
