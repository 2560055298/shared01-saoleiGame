HGE::Music_SetPos

设置样本列中的音乐位置。 

void Music_SetPos(
  HMUSIC music,
  int order,
  int row
);

参数

music
音乐句柄。
order
音乐样本集合中的实体编号。(order).
row
指定样本中的列编号row。

备注

可以使用Music_SetPos来通过order和row参数来设置音乐的播放位置。 调用Music_Play之后你还可以使用 Channel_SetPos来设置播放位置（以秒的方式）。

需要

头文件: hge.h
库文件: hge.lib

另见

Music_GetLength, Music_GetPos
