hgeFont::printfb

将格式化的文本字符串渲染到指定的矩形中。 

void printfb(
  float x,
  float y,
  float w,
  float h,
  int align,
  const char *format,
  ...
);

参数

x
矩形左上角X坐标。
y
矩形左上角Y坐标。
w
矩形的宽度。
h
矩形的高度。
align
排列分类符。可以使如下之一: HGETEXT_LEFT, HGETEXT_RIGHT, HGETEXT_CENTER 按位或者使用(|)连接如下任一个标记: HGETEXT_TOP, HGETEXT_BOTTOM, HGETEXT_MIDDLE.
format
C语言printf形式格式化规范。
...
要被渲染到屏幕上的可选参数列表。

备注

格式化规范的详细信息见编译器在线帮助或C语言参考手册。

需要

头文件: hgefont.h
库文件: hgehelp.lib

另见

Render, printfb
