hgeSprite::Render4V

将精灵渲染到屏幕上的任意quad中。 Renders the sprite to arbitrary quad on the screen. 

void Render4V(
  float x0,
  float y0,
  float x1,
  float y1,
  float x2,
  float y2,
  float x3,
  float y3
);

Parameters

x0
第一个顶点的X坐标。
y0
第一个顶点的Y坐标。
x1
第二个顶点的X坐标。
y1
第二个顶点的Y坐标。
x2
第三个顶点的X坐标。
y2
第三个顶点的Y坐标。
x3
第四个顶点的X坐标。
y3
第四个顶点的Y坐标。

需要

头文件: hgesprite.h
库文件: hgehelp.lib

另见

Render, RenderEx, RenderStretch
