1. 创建一个空的Win32 Application工程

打开File->New->Project...->Basic对话框标签.输入你的项目名并点击Windows Application图标:


点击Ok.

在下一个窗口中从你的工程中选择或者创建一个目录并点击Save:


2. 设置头文件和库文件路径

打开Project->Project Options->Directories对话框标签并选择Library Directories子标签. 点击...按钮,找到HGE“lib\gcc”目录并点击Ok。然后点击Add:


现在选择Include Directories子标签，并重复和对HGE“include”目录相同的操作：


现在不要点击Ok按钮。

3. 向工程中加入库文件

在相同的对话框中选择Parameters标签，点击Add Library or Object按钮，找到“hge.a”文件并点击Open按钮。 重复上面的操作将“hgehelp.a”加入：


点击Ok。

4. 开始编码！

包含hge.h头文件来访问HGE功能函数. 参考教学指导 部分了解初级编码的例子.
