hgeGUIObject数据成员

hgeGUIObject的数据成员是公有的，因此你可以直接访问它们： 

int      id;
bool     bStatic;
bool     bVisible;
bool     bEnabled;
hgeRect  rect;
DWORD    color;

hgeGUI       *gui;
hgeGUIObject *next;
hgeGUIObject *prev;

static HGE   *hge;

数据成员

id
你的构造函数必须将此字段设置为用于GUI的控件的标识。
bStatic
如果此字段为true，控件将无法获得焦点并且将无法接受键盘事件。 控件也会快速导航忽视。你必须在构造函数中初始化bStatic。
bVisible
如果此字段为true控件将被渲染到GUI，否则将无法看到它。 你必须在构造函数中初始化bVisible。
bEnabled
如果此字段为false控件将不会响应用户输入。 控件仍会接受信息，因此正确处理好该字段是你的任务。 你必须在构造函数中初始化bEnabled。
rect
此字段保存控件在屏幕坐标系上的边界框。 你必须在构造函数中初始化rect。
color
此字段保存控件的颜色。 你必须在构造函数中初始化color。
 
 
gui
这是一个指向使用该控件的GUI对象的指针。 当把控件加入GUI时，此字段由GUI初始化。 你可以使用gui来跟GUI交互，但是千万不能改变它。
next
这是一个内部字段，用于将所有的控件关联在一起。派生类千万不要修改它。
prev
这是一个内部字段，用于将所有的控件关联在一起。派生类千万不要修改它。
 
 
hge
这是一个使你可以在派生类中使用HGE接口的指针。

需要

头文件: hgegui.h
