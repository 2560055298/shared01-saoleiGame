使用HGE位图字体编辑器界面

基本信息和控制

当前字体纹理维数显示在屏幕左下角。 将鼠标指针移到一个字符上在屏幕右下角会显示它的参数。 鼠标左键拖动纹理来检查它当前在屏幕外的部分。

Escape键退出编辑器。

注意HGE字体由两个文件组成：一个描述和一个位图。

此外还需牢记一旦你生成了一个位图字体，你就可以使用图像编辑器对其附加任何效果。

字体参数

这组参数定义了字体的属性：

Typeface
系统安装了的字体列表。选择一个你想要在游戏中使用的字体。 要滚动列表可以使用滑块，鼠标滚轮或上/下方向键（当在列表内部单击后）。
Bold
是否使用粗体。
Italic
是否使用斜体。
Antialiased
是否对字符轮廓实施抗锯齿。 注意绝大多数字体有一个大小范围， 在此大小范围之外即使进行了抗锯齿处理也不会起作用。 这可能会在之后的版本中修正。 而现在你只能选择其他字体或者让该字体的稍微小一些/大一些来获得抗锯齿位图（当在HGE中使用时你可以将其恢复原状）。
Size
指定期望的字体高度，以像素为单位。
Characters range
指定字体中要包含的字母的范围。单击并拖动你需要的字母。

命令和信息

这里是描述编辑器各种信息和命令的地方:

Show grid
如果激活了这个复选框，将显示每个字母的边界框。 启用这个选项你将看到字符间会产生一像素的间隔。 这些都是源于HGE中允许字体的自由缩放的欠缺。
Save
打开标准的Windows另存为...对话框窗口。 选择一个路径和文件名来保存字体。 注意两个文件被保存：字体描述文件和字体位图文件。
Help
显示快速帮助。
Exit
退出编辑器。
