HGE::Gfx_SetClipping

为即将被渲染的所有图形设置剪裁区域。 

void Gfx_SetClipping(
  int x = 0,
  int y = 0,
  int w = 0,
  int h = 0
);

参数

x
剪裁区域的左上顶点x坐标。
y
剪裁区域左上顶点的y坐标。
w
剪裁区域的宽度。
h
剪裁区域的高度。

备注

如果所有参数被设置为0或者省略，剪裁区域将被重置为整个目标区域。当渲染目标转换为其他目标时，剪裁区域也会被设置为整个目标区域。

Gfx_SetClipping会冲掉所有之前积累的图形元（graphic primitives）。

需要

头文件: hge.h
库文件: hge.lib
