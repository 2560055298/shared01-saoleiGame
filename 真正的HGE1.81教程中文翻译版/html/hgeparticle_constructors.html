hgeParticleSystem构造函数

要创建并初始化一个hgeParticleSystem对象，你可以使用如下这些构造函数： 

hgeParticleSystem(
  const char *filename,
  hgeSprite *sprite
);

hgeParticleSystem(
  hgeParticleSystemInfo *psi
);

hgeParticleSystem(
  const hgeParticleSystem &ps
);

参数

filename
粒子系统描述文件名。此文件应该包含一个hgeParticleSystemInfo结构的磁盘镜像，用以描述粒子系统。
sprite
用于粒子系统的指向有效hgeSprite对象的指针。
psi
用以描述粒子系统的指向有效hgeParticleSystemInfo结构的指针。

备注

粒子系统描述文件通过Resource_Load函数加载。 因此将可以实用它的所有特性。

需要

头文件: hgeparticle.h
库文件: hgehelp.lib
