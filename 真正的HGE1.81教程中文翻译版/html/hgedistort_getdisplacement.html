hgeDistortionMesh::GetDisplacement

返回指定扭曲节点的平移量。 

void GetDisplacement(
  int col,
  int row,
  float *dx,
  float *dy,
  int ref
);

参数

col
网格节点的列。列数从0开始。
row
网格节点的行。行数从0开始。
dx
用于存储X轴平移量的指向float类型的指针。
dy
用于存储Y轴平移量的指向float类型的指针。
ref
平移量引用点。可以使以下任意一个：
HGEDISP_NODE - 引用点是默认节点位置
HGEDISP_TOPLEFT - 引用点是网格的左上角
HGEDISP_CENTER - 引用点是网格的中心

需要

头文件: hgedistort.h
库文件: hgehelp.lib

另见

SetDisplacement, GetZ, GetColor
