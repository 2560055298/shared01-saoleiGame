HGE::Music_Play

启动并播放一段音乐。 

HCHANNEL Music_Play(
  HMUSIC music,
  bool loop,
  int volume = 100,
  int order = -1,
  int row = -1
);

参数

music
要播放的音乐的句柄。
loop
如果为true音乐将被循环播放直至音频通道被停止。
如果为false音乐将只播放一次。
volume
初始音量，0..100。可选。
order
初始顺序。 -1为使用当前值。可选。
row
初始行。 -1为使用当前值。可选。

返回值

返回播放音乐的音频通道句柄。 可以用其调用音频通道控制函数。

备注

一次只能有最多1个实例运行。 如果音乐已经在播放状态中， 那么它将以默认参数重新开始。

需要

头文件: hge.h
库文件: hge.lib

另见

Music_Load, Music_Free
