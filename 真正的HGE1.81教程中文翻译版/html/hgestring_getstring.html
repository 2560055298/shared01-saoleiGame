hgeStringTable::GetString

返回由名称指定的字符串的指针。 

char *GetString(
  const char *name
);

参数

name
字符串名。

返回值

如果执行成功则返回指向字符串的指针，否则返回0。

备注

一旦字符串表文件在hgeStringTable构造函数中被解析， GetString将返回指向该字符串的内部指针。每个字符串只维护一个副本。 每次以指定的字符串名称调用GetString所返回的指针是不变的， 因此可以在你的代码中保存该指针。返回的指针在调用hgeStringTable析构函数之前是有效的。

你不应该释放掉返回的指针。 所有的字符串内存都是通过调用hgeStringTable析构函数来释放的。

需要

头文件: hgestrings.h
库文件: hgehelp.lib
