hgeCreate函数

返回一个指向HGE接口的指针，如果HGE对象不存在则会先创建它。 

HGE *hgeCreate(
  int ver
);

参数

ver
HGE API 版本号。这里传递HGE_VERSION常量即可。

返回值

如果加载的HGE.DLL中存在符合的API，hgeCreate将返回一个指向HGE接口的指针，否则返回0。

备注

hgeCreate是一个全局的C语言函数。 它位于所有类之外，通过引用HGE 动态链接库加载。

要访问HGE函数你需要由hgeCreate函数返回的指向HGE接口的指针 To access HGE functions you need the HGE interface pointer that is returned by hgeCreate function. hgeCreate在必要时(没有可用的HGE对象时)会创建HGE对象，并将内部的引用计数器的值相应的增加。 hgeCreate creates a HGE object if needed and increases the internal reference counter. 要想释放到已获得的接口你必须调用Release函数

你可以在你需要访问HGE的时候调用hgeCreate创建它， 然后通过调用Release释放掉它的资源。 或者你可以再程序启动时创建一次该接口，并将它存储在一个全局变量中，然后在程序结束释放资源时解决它。 虽然后者被认为是一个不好的做法，但是在程序不是太大的时候，不会出现问题。
