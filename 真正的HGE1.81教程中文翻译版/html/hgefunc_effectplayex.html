HGE::Effect_PlayEx

启动并播放一个声效，可以设置额外的参数。 

HCHANNEL Effect_PlayEx(
  HEFFECT effect,
  int volume = 100,
  int pan = 0,
  float pitch = 1.0,
  bool loop = false
);

参数

effect
要播放的声效句柄。
volume
可选的声效音量0~100，默认为100，即最大音量。
pan
可选的声效平衡(pan)，-100 ~ 100，默认为0，即中间值。
pitch
可选的音效音高乘数。默认是1.0，即默认音高。Optional effect pitch multiplier. Default is 1.0, meaning default pitch.
loop
可选参数，如果为true，音效会一直循环播放直至音频通道被关闭，默认值是false。

返回值

返回播放声效的音频通道句柄。 可以用其调用音频通道控制函数。

Remarks

一次只能有最多4个实例同事运行。 如果调用Effect_PlayEx时所有4个进程都在使用， 那么音量最小的进程将以新的参数重新运行。

需要

头文件: hge.h
库文件: hge.lib
