HGE::Resource_MakePath

创建一个绝对文件路径。 Builds absolute file path. 

char *Resource_MakePath(
  const char *filename = 0
);

参数

filename
相对或绝对路径，或0。

返回值

如果filename是相对路径，则将其当做相对于程序目录的路径并返回从磁盘根目录开始的完全路径。 如果filename是绝对路径，则将其拷贝到临时缓存，并返回指向该缓存的指针。 如果filename等于0或者被省略，则返回程序所在文件夹的绝对路径跟着一个“\”。

需要

头文件: hge.h
库文件: hge.lib

另见

Resource_EnumFiles, Resource_EnumFolders
