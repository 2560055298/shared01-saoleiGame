hgeResourceManager::GetParticleSystem

返回指向一个已命名的粒子系统的指针。 

hgeParticleSystem *GetParticleSystem(
  const char *name
);

参数

name
粒子系统名称。

返回值

如果成功，返回指向hgeParticleSystem对象的指针，否则返回0。

备注

要通过hgeResourceManager使用粒子系统，必须在资源脚本中定义它。

如果粒子系统没有被创建，将先创建它。否则将返回指向已存在粒子系统句柄的指针。

你可以使用返回的粒子系统的info成员通过hgeParticleManager来生成该对象的多个实例。

需要

头文件: hgeresource.h
库文件: hgehelp.lib
