hgeDistortionMesh::SetDisplacement

为指定的扭曲节点设置平移量。 

void SetDisplacement(
  int col,
  int row,
  float dx,
  float dy,
  int ref
);

参数

col
要改变的节点的列。列数字从0开始。
row
要改变的节点的行。行数字从0开始。
dx
X轴平移量。
dy
Y轴平移量。
ref
平移量引用点。可以使以下任意一个：
HGEDISP_NODE - 引用点是默认节点位置
HGEDISP_TOPLEFT - 引用点是网格的左上角
HGEDISP_CENTER - 引用点是网格的中心

需要

头文件: hgedistort.h
库文件: hgehelp.lib

另见

GetDisplacement, SetZ, SetColor
