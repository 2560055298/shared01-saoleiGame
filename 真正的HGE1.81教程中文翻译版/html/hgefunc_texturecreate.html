HGE::Texture_Create

创建一个空纹理。 

HTEXTURE Texture_Create(
  int width,
  int height
);

参数

width
要创建的纹理的宽度。
height
要创建的纹理的高度。

返回值

如果执行成功，返回纹理句柄。 否则返回0.

备注

渲染目标维数必须是2的幂数，如果此条件不满足，请求的维数将被自动转换为最接近的2的幂数值的维数，大于请求的维数。

你可以使用Texture_Lock 和 Texture_Unlock函数 来给纹理填充一些数据。

当纹理不再被使用时，必须使用Texture_Free函数来删除它。

需要

头文件: hge.h
库文件: hge.lib

另见

Texture_Load, Texture_Free, Texture_Lock, Texture_Unlock
