HGE::Target_Create

创建一个渲染目标。 

HTARGET Target_Create(
  int width,
  int height,
  bool zbuffer
);

参数

width
要创建目标的宽度。
height
要创建目标的高度。
zbuffer
如果为true, 将建立一个Z缓冲用以渲染目标。

返回值

如果执行成功，返回供Gfx_BeginScene使用的渲染目标句柄 否则返回0。

备注

渲染目标维数必须是2的幂数，如果此条件不满足，请求的维数将被自动转换为最接近的2的幂数值的维数，大于请求的维数。

渲染目标只支持32位视频模式下的透明通道，16位视频模式下渲染目标总是不透明的。可以查看HGE_SCREENBPP 了解当前运行的是何种视频模式。

如果函数调用失败，可以使用System_GetErrorMessage函数获得错误信息。 另外如果设置了HGE_LOGFILE系统标记，错误信息将被写入日志。

当渲染目标不再被使用时，必须使用Target_Free来删除它。

需要
