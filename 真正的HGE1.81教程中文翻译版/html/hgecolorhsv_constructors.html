hgeColorHSV Constructors

要创建并初始化一个hgeColorHSV对象你可以使用如下构造函数之一： 

hgeColorHSV();

hgeColorHSV(
  DWORD color
);

hgeColorHSV(
  float h,
  float s,
  float v,
  float a
);

hgeColorHSV(
  const hgeColorHSV &col
);

参数

color
用于初始化hgeColorHSV对象的硬件颜色。
h
色调值范围从0.0到1.0。
s
饱和度值范围从0.0到1.0。
v
亮度值范围从0.0到1.0。
a
透明度值范围从0.0到1.0。

需要

头文件: hgecolor.h
