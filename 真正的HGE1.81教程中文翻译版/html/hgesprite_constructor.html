hgeSprite 构造函数

要创建和初始化一个hgeSprite对象，可以使用如下任意构造函数： 

hgeSprite(
  HTEXTURE tex,
  float x,
  float y,
  float w,
  float h
);

hgeSprite(
  const hgeSprite &sprite
);

参数

tex
纹理句柄，保存精灵图像或者0。
x
精灵纹理的X坐标（以纹理坐标的形式即0.0到1.0）。
y
精灵纹理的Y坐标（以纹理坐标的形式即0.0到1.0）。
w
精灵宽度。
h
精灵高度。

备注

如果tex = 0，将使用白色作为纹理数据。

需要

头文件: hgesprite.h
库文件: hgehelp.lib
