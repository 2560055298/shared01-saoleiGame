hgeGUIListbox类

hgeGUIListbox是一个用于列表框GUI控件的类。 要创建一个hgeGUIListbox对象可以使用如下构造函数： 

hgeGUIListbox(
  int       id,
  float     x,
  float     y,
  float     w,
  float     h,
  hgeFont  *fnt,
  DWORD     tColor,
  DWORD     thColor,
  DWORD     hColor
);

参数

id
已创建控件的整型标识。
x
列表框的左上角X坐标。
y
列表框的左上角Y坐标。
w
列表框的宽度。
h
列表框的高度。
fnt
用于渲染列表框项文本的指向hgeFont对象的指针。
tColor
用于渲染列表框项的颜色。
thColor
用于渲染高亮列表框项的颜色。
hColor
用于渲染高亮列表框项条纹的颜色。

其他方法

AddItem 向列表框中加入一项
DeleteItem 从列表框中删除一项
GetSelectedItem 返回当前选中项的索引。
SetSelectedItem 设置选中项.
GetTopItem 返回显示的第一项的索引。
SetTopItem 将指定项设置为显示的第一项。
  
GetItemText 返回指定项的文本字符串。
GetNumItems 返回列表框中所有项的数目。
GetNumRows 返回列表框中同时可显示的项数。
Clear 从列表框中移除所有项。

需要

头文件: hgeguictrls.h
库文件: hgehelp.lib
