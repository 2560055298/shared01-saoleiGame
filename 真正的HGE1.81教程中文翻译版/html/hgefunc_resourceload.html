HGE::Resource_Load

从资源包或磁盘上加载一个资源到内存中。 

void *Resource_Load(
  const char *filename,
  DWORD *size = 0
);

参数

filename
资源文件名
size
DWORD指针用来存储资源大小。如果等于0或者被省略，Resource_Load将不会返回资源的大小。

返回值

如果成功，返回一个指针，指向为该资源分配的内存块，并将资源大小保存在size指向的DWORD类型指针中。 否则返回0.

备注

如果filename是一个相对路径，则会通过Resource_AttachPack在所有资源包范围内搜索该资源， 然后再在程序文件夹查找。 如果filename是一个绝对路径，则会直接从该磁盘路径加载资源文件。

如果函数调用失败，你可以使用System_GetErrorMessage函数获得错误信息。 如果设置了系统状态HGE_LOGFILE，错误信息将被写入日志。

当资源不再有用时，你必须使用Resource_Free释放掉已分配的内存资源。
