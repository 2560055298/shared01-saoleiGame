HGE 输入事件（Input Event）结构

输入事件结构结合Input_GetEvent函数使用来获得缓冲的输入事件： 

struct hgeInputEvent
{
  int    type;
  int    key;
  int    flags;
  int    chr;
  int    wheel;
  float  x;
  float  y;
};

成员

type
事件类型，可以是如下之一：
INPUT_KEYDOWN - 键盘按键按下
INPUT_KEYUP - 键盘按键松开
INPUT_MBUTTONDOWN - 鼠标按键按下
INPUT_MBUTTONUP - 鼠标按键松开
INPUT_MOUSEMOVE - 鼠标移动
INPUT_MOUSEWHEEL - 鼠标滚轮滚动
key
键值，只对事件 INPUT_KEYDOWN, INPUT_KEYUP, INPUT_MBUTTONDOWN and INPUT_MBUTTONUP有效。
flags
按位或使用（|）连接如下常量，来指定事件上下文： HGEINP_SHIFT - 任意shift键按下
HGEINP_CTRL - 任意ctrl键按下
HGEINP_ALT - 任意alt键按下
HGEINP_CAPSLOCK - 大写锁定(Caps Lock)启用
HGEINP_NUMLOCK - 小键盘数字锁定（Num Lock）启用
HGEINP_SCROLLLOCK - 滚动锁定（Scroll Lock）启用
HGEINP_REPEAT - 对于INPUT_KEYDOWN:此事件由自动重复特性产生； 对于INPUT_MBUTTONDOWN此事件是双击
chr
字符代码，表示当前键盘状态和输入地域。只对INPUT_KEYDOWN和INPUT_KEYUP时间有效。
wheel
鼠标滚轮滚动的刻度数。 正值表示滚轮向前滚动，远离用户； 负值表示滚轮向后滚动，朝向用户。 只对事件INPUT_MOUSEWHEEL有效。
x
鼠标指针的X坐标。
y
鼠标指针的Y坐标。
