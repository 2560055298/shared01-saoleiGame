hgeAnimation::SetMode

设置动画回放模式。 

void SetMode(
  int mode
);

参数

mode
按位或者使用(|)连接回放常量值：
HGEANIM_FWD - 向前播放动画。
HGEANIM_REV - 反向播放动画。
HGEANIM_PINGPONG - 使用乒乓循环(ping-pong cycle)
HGEANIM_NOPINGPONG - 不使用乒乓循环(ping-pong cycle)
HGEANIM_LOOP - 循环播放动画。
HGEANIM_NOLOOP - 不循环播放动画。

备注

默认的动画模式是HGEANIM_FWD | HGEANIM_LOOP.

需要

头文件: hgeanim.h
库文件: hgehelp.lib
