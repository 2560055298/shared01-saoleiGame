hgeFont::Render

将文本字符串渲染到屏幕上。 

void Render(
  float x,
  float y,
  int align,
  const char *string
);

参数

x
屏幕X坐标。
y
屏幕Y坐标。
align
排列分类符。可以使如下之一: HGETEXT_LEFT, HGETEXT_RIGHT, HGETEXT_CENTER.
string
要渲染的字符串。

备注

换行符('\n')可以被正确处理。所有其他负号将被看做普通字母处理。 字体中未定义的符号将被渲染为'?'。
