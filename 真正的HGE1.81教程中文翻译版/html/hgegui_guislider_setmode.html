hgeGUISlider::SetMode

修改滑动条的行为模式。 

void SetMode(
  float fMin,
  float fMax,
  int   mode
);

参数

fMin
滑动条允许的最小值。
fMax
滑动条允许的最大值。
mode
滑动条的可视化模式，可以使以下之一：
  • HGESLIDER_BAR
    滑动条显示为起始于滑动条的最左点终止于当前滑动条位置的条状物。滑动条的图像是拉伸的， 因此，要避免显示时扭曲，需要确保其在每个方向上的指定区纹理域外多延伸一个像素。
  • HGESLIDER_BARRELATIVE
    滑动条显示为起始于滑动条中心终止于当前滑动条位置的条状物。滑动条的图像是拉伸的， 因此，要避免显示时扭曲，需要确保其在每个方向上的指定区纹理域外多延伸一个像素。
  • HGESLIDER_SLIDER
    滑动条显示为一个固定大小的头标，放置位置取决于当前滑动条的值。

需要

头文件: hgeguictrls.h
库文件: hgehelp.lib
