HGE::Effect_Load

从内存，资源包或磁盘中加载一个声效(Sound Effect)。 

HEFFECT Effect_Load(
  const char *filename,
  DWORD size = 0
);

参数

filename
声效文件名
size
如果此参数不是0，那么它将是包含任意有效格式声效的内存块的大小 而filename则被认为是指向此内存块的指针。

返回值

成功则返回可被函数Effect_Play 和 Effect_PlayEx使用的声效句柄。 否则返回0。

备注

支持 WAV, MP3, MP2, MP1 and OGG 音频格式。

如果filename是一个相对路径，则会使用Resource_AttachPack先从所有资源包中检索并 附加文件，然后是程序目录。 如果filename是一个绝对路径，将直接把声音文件通过指定的路径从磁盘载入。

如果函数调用失败，可是通过调用System_GetErrorMessage函数获得错误信息。 如果设置了系统标记HGE_LOGFILE，错误信息将被写入日志文件。

当声效不再被使用时，必须使用Effect_Free函数删除它。

需要

头文件: hge.h
库文件: hge.lib

另见

Effect_Free, Effect_Play, Effect_PlayEx
