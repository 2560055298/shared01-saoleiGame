hgeGUIButton类

hgeGUIButton是一个按钮GUI控件类。要创建一个hgeGUIButton对象，可以使用如下构造函数： 

hgeGUIButton(
  int       id,
  float     x,
  float     y,
  float     w,
  float     h,
  HTEXTURE  tex,
  float     tx,
  float     ty
);

参数

id
要创建的控件的整型标识。
x
按钮左上角X坐标。
y
按钮左上角Y坐标。
w
按钮的宽度。
h
按钮的高度。
tex
纹理句柄。保存按钮的图像。 纹理必须保存两个图像，一个是按钮未按下状态的图像，紧接着该图像右边的是该按钮按下状态时的图像。
tx
按钮图像左上角纹理x坐标，以像素为单位。
ty
按钮图像左上角纹理y坐标，以像素为单位。

Other methods

SetMode 设置按钮的行为模式。
SetState 设置按钮的状态。
GetState 返回按钮的状态。

需要

头文件: hgeguictrls.h
库文件: hgehelp.lib
