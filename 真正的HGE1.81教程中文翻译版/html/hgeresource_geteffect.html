hgeResourceManager::GetEffect

返回一个已命名的声效句柄。 

HEFFECT GetEffect(
  const char *name,
  int resgroup = 0
);

参数

name
声效名称。
resgroup
资源组标识。

返回值

如果成功，则返回声效句柄，否则返回0。

备注

你可以使用资源脚本中定义的名称或者直接使用一个纯文件名， 一个程序文件夹或资源包根目录的绝对或相对路径。

如果声效没有被加载，将先加载声效。否则将返回指向已存在声效的指针。

参数resgroup只有在指定了纯文件名时才会使用。 当加载资源脚本中定义的资源时，resgroup从脚本中获得。 使用这里通过Purge传递的值来释放资源，如果=0或者省略- 将不会有指定的组被分配。

需要

头文件: hgeresource.h
库文件: hgehelp.lib
