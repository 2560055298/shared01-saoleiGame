HGE::Gfx_RenderTriple

渲染一个Triple。 

void Gfx_RenderTriple(
  const hgeTriple *triple
);

参数

triple
指向hgeTriple结构的指针，用来描述要被渲染的Triple。

备注

Gfx_RenderTriple函数必须在Gfx_BeginScene和Gfx_EndScene 函数之间调用。

Triple是基本的HGE图形基本类型。更多Triple的信息请参考Triple结构体部分。 如果Triple的tex字段被设置为0，白色将被用作纹理数据。 还需要注意的是triple的三个顶点都可以有它们自己的颜色。透明度和Z顺序(深度)。 triple不需要做任何额外的屏幕和纹理坐标优化便可以自动调整与quad的接合渲染。

考虑到效率问题Gfx_RenderTriple积攒连续的若干含有相同的纹理和混合模式的triple并使用一个DirectX调用来渲染它们。 绝大多数时候你不会发现其中的不同，并且也没有必要去处理这些批处理操作。 但是为了保证准确和安全，建议还是通过纹理和混合模式来组织好你的Gfx_RenderTriple函数调用。
