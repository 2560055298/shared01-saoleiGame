hgeGUIText类

hgeGUIText是一个用于文本GUI控件的类。要创建一个hgeGUIText对象可以使用如下构造函数： 

hgeGUIText(
  int       id,
  float     x,
  float     y,
  float     w,
  float     h,
  hgeFont*  fnt
);

参数

id
要创建的控件的整型标识。
x
控件的左上角X坐标。
y
控件的左上角Y坐标。
w
控件的宽度。
h
控件的高度。
fnt
用于渲染文本的指向hgeFont对象的指针。

其他方法

SetMode 设置文本排列模式。
SetText 设置控件文本。
printf 向空间的文本内输出格式化字符串。
SetColor 设置文本的颜色。

需要

头文件: hgeguictrls.h
库文件: hgehelp.lib
